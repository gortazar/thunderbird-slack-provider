Authentication

Enter a Slack Bot Token (starts with xoxb-) or User Token (starts with xoxp-) below. See the README for step-by-step instructions on creating a Slack app and obtaining a token.

Connection Status

● Unknown — use "Test Connection" to verify.

Display

When enabled, user profile pictures are never loaded and each message shows the author's initials (e.g. JD for John Doe).

Required Scopes

Your Slack app / token must have the following OAuth scopes granted:

  • channels:read – list public channels
  • groups:read – list private channels you're a member of
  • channels:history – read messages in public channels
  • groups:history – read messages in private channels
  • chat:write – send messages
  • users:read – resolve user names and avatars
  • channels:manage (optional) – mark channels as read
